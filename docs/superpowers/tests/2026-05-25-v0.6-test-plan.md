### 1. Token 持久化 + 自动登录

| 步骤 | 操作窗口 | 操作 | 预期结果 |
|------|---------|------|----------|
| 1.1 | Client A | `/register alice123 Alice` | 注册成功，显示 QQ 号（如 10001） |
| 1.2 | Client B | `/register bob456 Bob` | 注册成功，显示 QQ 号（如 10002） |
| 1.3 | Client A | `/quit` | 退出 |
| 1.4 | Client A | `go run ./cmd/client` | 重新启动客户端 |
| 1.5 | Client A | 观察启动后输出 | 应显示 `[cmd] found saved token for QQ:10001, auto-login...` |
| 1.6 | Client A | 观察登录结果 | 应显示 `login ok, Alice(QQ:10001)` |
