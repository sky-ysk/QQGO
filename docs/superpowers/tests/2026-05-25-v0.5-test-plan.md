# v0.5 手动测试方案

**分支：** `feature/v0.5`
**日期：** 2026-05-25

---

## 准备工作

```bash
# 1. 确认分支
cd /Users/yangshikang.6/Desktop/Code/Go/QQGO
git branch --show-current   # 应显示 feature/v0.5

# 2. 删除旧数据库
rm -f qqgo.db

# 3. 编译
go build ./...

# 4. 启动服务端
go run ./cmd/server

# 5. 打开 3 个终端，分别启动客户端
# 终端 A:
go run ./cmd/client
# 终端 B:
go run ./cmd/client
# 终端 C:
go run ./cmd/client
```

## 测试项

### 1. 非好友消息限制

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1.1 | 终端A: `/register alice123 Alice` | 注册成功，得到 QQ 号（如 10001） |
| 1.2 | 终端B: `/register bob456 Bob` | 注册成功，得到 QQ 号（如 10002） |
| 1.3 | 终端A: `/to 10002` | 切换成功 |
| 1.4 | 终端A: 发送 "hello" | `[sent ✓] ok`，终端B 收到消息 |
| 1.5 | 终端A: 发送 "second msg" | `[sent ✓] not friend, only 1 message allowed` |
| 1.6 | 终端A: `/addfriend 10002 你好` | 发送好友请求 |
| 1.7 | 终端B: `/accept 10001` | 接受好友 |
| 1.8 | 终端A: 发送 "hello again" | `[sent ✓] ok`，正常发送 |
| 1.9 | 终端A: 发送 "another" | `[sent ✓] ok`，好友无限制 |
| 1.10 | 终端A: `/delfriend 10002` | 删除好友 |

| 1.12 | 终端A: 发送 "second after delete" | `[sent ✓] not friend, only 1 message allowed` |

## 已知边界情况

- 非好友消息计数是单向的（A→B 和 B→A 独立计数）
- 群消息历史记录暂不支持翻页（`/to` 只对私聊有效）
- 会话列表只显示有过消息往来的会话
- 群主退出群会报错，需先转让群主（功能未实现）
