# v0.4 测试方法

**版本：** v0.4
**分支：** `feature/v0.4-qq-identity`
**日期：** 2026-05-22
**类型：** 全系统重构（QQ 号作为唯一标识）

---

## 准备工作

```bash
# 1. 确认在 feature/v0.4-qq-identity 分支
git branch --show-current

# 2. 删除旧数据库（字段类型变更，必须重建）
rm -f qqgo.db

# 3. 启动服务端
go run ./cmd/server

# 4. 打开 2-3 个终端，启动客户端（无需参数）
# 终端 A:
go run ./cmd/client

# 终端 B:
go run ./cmd/client
```

---

## 测试项

### 1. 注册与登录

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1.1 | 终端A: `/register alice123 Alice` | 注册成功，自动登录，显示 QQ 号（如 10001） |
| 1.2 | 终端A: `/whoami` | 显示 `QQ: 10001` |
| 1.3 | 终端B: `/register bob456 Bob` | 注册成功，自动登录，显示 QQ 号（如 10002） |
| 1.4 | 终端B: `/quit`，重新 `go run ./cmd/client` | 新客户端启动 |
| 1.5 | 终端B: `/login 10002 bob456` | 登录成功 |
| 1.6 | 终端B: `/login 10002 wrongpassword` | 报错 "auth failed"，连接保持 |
| 1.7 | 终端B: `/whoami` | 仍显示 QQ: 10002（登录失败未切换） |

### 2. 点对点消息

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 2.1 | 终端A: `/to 10002` | 切换目标成功 |
| 2.2 | 终端A: 发送 "hello bob" | 收到 `[sent ✓] ok`，终端B 收到消息 |
| 2.3 | 终端B: 看到消息 | 显示 `[10001 -> 10002]: hello bob` |
| 2.4 | 终端A: `/to 99999` | 切换目标成功 |
| 2.5 | 终端A: 发送 "hello nobody" | 收到 `[sent ✓] user not found`（BUG-007） |

### 3. 好友管理

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 3.1 | 终端A: `/addfriend 10002 你好` | 发送好友请求 |
| 3.2 | 终端B: 收到通知 | `[Friend Request] ...` |
| 3.3 | 终端B: `/friends` | 待处理列表显示 Alice |
| 3.4 | 终端B: `/accept 10001` | 接受成功 |
| 3.5 | 终端A: `/friends` | 我的好友列表显示 Bob |
| 3.6 | 终端A: `/search Bob` | 搜索结果包含 Bob |
| 3.7 | 终端A: `/search 10002` | 搜索结果包含 Bob（QQ 号精确匹配） |
| 3.8 | 终端A: `/remark 10002 同事小王` | 备注设置成功 |
| 3.9 | 终端A: `/friends` | Bob 显示为 `同事小王(Bob)` |
| 3.10 | 终端A: `/delfriend 10002` | 删除好友成功 |

### 4. 好友分组管理

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 4.1 | 终端A: 重新加 Bob 为好友 | 恢复好友关系 |
| 4.2 | 终端A: `/creategroup 家人` | 成功 |
| 4.3 | 终端A: `/groups` | 列表包含"我的好友"和"家人" |
| 4.4 | 终端A: `/creategroup 家人` | 报错 "group already exists" |
| 4.5 | 终端A: `/creategroup 我的好友` | 报错 "invalid group name" |
| 4.6 | 终端A: `/movefriend 10002 同事` | 报错 "group not found" |
| 4.7 | 终端A: `/creategroup 同事` → `/movefriend 10002 同事` | 成功 |
| 4.8 | 终端A: `/friends` | Bob 显示在"同事"分组 |
| 4.9 | 终端A: `/delgroup 同事`（组内有 Bob） | 报错 "group is not empty" |
| 4.10 | 终端A: `/movefriend 10002 我的好友` → `/delgroup 同事` | 成功删除 |
| 4.11 | 终端A: `/delgroup 我的好友` | 报错 "cannot delete default group" |
| 4.12 | 终端A: `/creategroup 空组` → `/groups` | "空组"出现在列表中 |

### 5. 优雅退出（BUG-001/008）

| 5.3 | 终端B: `/quit` | 正常退出，**无** `read error` 日志 |

### 6. 离线消息
